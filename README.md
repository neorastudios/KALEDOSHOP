# Kaledoshop 🦅

**L'esprit calédonien chez vous**

E-commerce calédonien — Produits malins pour améliorer le quotidien.

## Structure

```
kaledoshop/
├── index.html          ← Page d'accueil
├── css/
│   └── style.css       ← Styles (Barely There UI)
```
